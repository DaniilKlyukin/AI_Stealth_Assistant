# Stealth Assistant

**Stealth Assistant** — это компактный скрипт на Python, предназначенный для полуавтоматического анализа текста с экрана или из буфера обмена с последующим получением ответов от локальной языковой модели (LLM). 

Программа работает по следующему принципу:
1. Вы выделяете область экрана с текстом (например, тест, вопрос или статью).
2. Скрипт делает снимок этой области, распознает текст с помощью библиотеки EasyOCR (поддерживаются русский и английский языки).
3. Распознанный текст отправляется на локально запущенный сервер `llama.cpp`.
4. Полученный ответ отображается в небольшом полупрозрачном окне поверх всех остальных окон. Окно можно свободно перемещать по экрану мышкой или мгновенно скрывать.

## Подготовка к работе и системные требования

Для запуска программы вам понадобятся:
1. **Интерпретатор Python** (рекомендуется версия от 3.8 до 3.11, так как некоторые библиотеки, такие как EasyOCR и PyTorch, могут иметь проблемы с совместимостью на более новых версиях).
2. **Пакет llama.cpp** для локального запуска языковой модели.
3. **Модель в формате GGUF** (например, Qwen, Llama или Mistral).

### Шаг 1. Установка Python и зависимостей

1. Установите Python, обязательно отметив галочку **"Add Python to PATH"** при установке.
2. Установите необходимые библиотеки. Откройте терминал (командную строку) и выполните команду:
   ```bash
   pip install opencv-python numpy mss keyboard easyocr requests python-dotenv
   ```
   *Примечание: При первом запуске библиотека EasyOCR автоматически скачает небольшие файлы моделей для распознавания текста (русского и английского). Это может занять некоторое время.*

### Шаг 2. Загрузка llama.cpp

1. Перейдите в репозиторий проекта на GitHub: [ggerganov/llama.cpp/releases](https://github.com/ggerganov/llama.cpp/releases).
2. Скачайте подходящий архив для вашей операционной системы. Для Windows с видеокартами Nvidia обычно выбирают версию с поддержкой CUDA (например, `llama-...-bin-win-cuda-x64.zip`), для карт AMD — Vulkan, а для систем без дискретной видеокарты — базовую версию `cpu`.
3. Распакуйте архив в удобную для вас папку (например, `E:\games\llama-server\`). Внутри должен находиться исполняемый файл `llama-server.exe` (в старых версиях он мог называться `server.exe`).

### Шаг 3. Загрузка языковой модели (LLM)

1. Модели распространяются в формате `.gguf`. Популярным выбором для работы с русским и английским языками являются модели семейства **Qwen** (например, `Qwen2.5-7B-Instruct` или `Qwen3.5-9B`) или **Llama-3**.
2. Скачать их можно на платформе Hugging Face. Выберите квантованную версию (например, с пометкой `Q4_K_M` или `Q5_K_M` — они представляют собой разумный баланс между размером, скоростью работы и качеством ответов).
3. Сохраните файл модели на вашем диске (например, `E:\AI_Models\Qwen3.5-9B.Q4_K_M.gguf`).

## Настройка конфигурационного файла `.env`

Создайте в той же папке, где находится файл `stealth_assistant.py`, текстовый файл с именем `.env` (обратите внимание на точку в начале названия). Скопируйте туда следующие параметры и укажите свои пути к файлам:

```ini
# URL-адрес запущенного сервера llama.cpp
API_BASE_URL=http://127.0.0.1:8080

# Идентификатор модели (можно указать имя файла)
LLM_MODEL=Qwen3.5-9B.Q4_K_M

# Системный промпт, который указывает модели, как именно отвечать на текст
PROMPT="На экране тест. Дай правильный ответ (кратко)."

# Время ожидания ответа от сервера в секундах
API_TIMEOUT=300

# Начальные координаты окна с ответом на экране
OVERLAY_X=50
OVERLAY_Y=600

# Горячие клавиши для управления
HOTKEY_SELECT=ctrl+f8
HOTKEY_MANUAL=ctrl+f9
HOTKEY_TOGGLE_OVERLAY=ctrl+f10
HOTKEY_CLIPBOARD=ctrl+f11

# Автоматический запуск сервера llama.cpp вместе со скриптом ассистента
AUTO_START_SERVER=True

# Полный путь к исполняемому файлу сервера llama.cpp
SERVER_EXE_PATH=E:\games\llama-b8868-bin-win-vulkan-x64\llama-server.exe

# Полный путь к файлу модели GGUF
MODEL_PATH=E:\AI_Models\Qwen3.5-9B.Q4_K_M.gguf

# Размер контекстного окна (в токенах)
CONTEXT_SIZE=8192

# Количество слоев модели, отправляемых на видеокарту (GPU). 
# Значение 99 пытается перенести все слои. Уменьшите, если не хватает видеопамяти.
GPU_LAYERS=99
```

## Инструкция по использованию

### Запуск программы

Запустите скрипт через командную строку от имени администратора (это требуется библиотеке `keyboard` для перехвата глобальных горячих клавиш, когда окно консоли не в фокусе):
```bash
python stealth_assistant.py
```
После запуска на экране появится небольшое окно с текстом «Программа запущена!». Если в параметре `AUTO_START_SERVER` указано `True`, скрипт также фоном запустит процесс `llama-server.exe`.

### Управление с помощью горячих клавиш

*   **`Ctrl + F8` (Выбор области экрана):** Экран затеняется. Зажмите левую кнопку мыши и выделите область, в которой обычно появляются вопросы или тексты. Выделенная область сохранится в памяти программы.
*   **`Ctrl + F9` (Анализ экрана):** Скрипт сделает снимок ранее выбранной области, распознает текст с помощью OCR, отправит его на локальный сервер LLM и выведет ответ в окно-оверлей. В процессе обработки в окне будет отображаться надпись `[Думаю...]`.
*   **`Ctrl + F11` (Анализ из буфера обмена):** Если текст трудно распознать через OCR, вы можете выделить его мышкой, скопировать в буфер обмена (`Ctrl + C`) и нажать `Ctrl + F11`. Скрипт отправит скопированный текст напрямую в LLM без создания скриншота.
*   **`Ctrl + F10` (Скрыть/Показать окно):** Быстро скрывает или возвращает окно с ответами на экран.

### Управление окном вывода ответов

*   **Перемещение:** Вы можете перетащить окно с ответом в любое удобное место экрана. Для этого зажмите левую кнопку мыши на тексте внутри окна и перетащите его.
*   **Закрытие программы:** Для завершения работы закройте консоль, в которой запущен скрипт. При этом процесс локального сервера `llama.cpp` завершится автоматически.
